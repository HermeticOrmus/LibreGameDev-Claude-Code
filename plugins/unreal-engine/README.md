- **Unreal Developer** - Specialized agent for Blueprints, C++ Unreal, Gameplay Framework, materials

### Commands
- `/unreal` - Quick-access command for unreal-engine workflows

### Skills
- **Unreal Patterns** - Pattern library and knowledge base for unreal-engine

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/unreal analyze

# Use the command with specific input
/unreal generate --context "your project"

# Reference patterns from the skill
"Apply unreal-patterns patterns to this implementation"
```
