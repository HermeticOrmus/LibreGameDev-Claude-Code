# Asset Pipelines

Asset import, optimization, atlasing, LOD generation

## What's Included

### Agents
- **Asset Pipeline Engineer** - Specialized agent for Asset import, optimization, atlasing, LOD generation

### Commands
- `/assets` - Quick-access command for asset-pipelines workflows

### Skills
- **Asset Pipeline Patterns** - Pattern library and knowledge base for asset-pipelines

## Quick Start

1. Copy this plugin to your Claude Code plugins directory
2. Use the agent for guided, multi-step workflows
3. Use the command for quick, targeted operations
4. Reference the skill for patterns and best practices

## Usage Examples

```
# Use the command directly
/assets analyze

# Use the command with specific input
/assets generate --context "your project"

# Reference patterns from the skill
"Apply asset-pipeline-patterns patterns to this implementation"
```

## Key Patterns

- Follow established conventions for asset-pipelines
- Validate inputs before processing
- Document decisions and rationale
- Test outputs against requirements
- Iterate based on feedback
