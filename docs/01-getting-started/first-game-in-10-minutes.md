By the end of this tutorial, you'll have a complete Pong game featuring:

- **Two Paddles**: Player-controlled (left) and AI-controlled (right)
- **Ball Physics**: Realistic bouncing with angle-based reflection

- **AI Opponent**: Adaptive AI that tracks and intercepts the ball

### Step 2: Generate Basic Game Structure

Now we'll use Claude Code to generate our complete game.

**The Initial Prompt**:

Open Claude Code (web interface or CLI) and enter this exact prompt:

```
Create a complete Pong game in a single HTML file using HTML5 Canvas and vanilla JavaScript. Include:

1. Two paddles: left paddle (player-controlled with W/S keys), right paddle (AI-controlled)
2. A ball that bounces off paddles and top/bottom walls
3. Ball resets to center when it goes off left or right edges
4. Score tracking for both players (displayed on screen)
5. AI that follows the ball's Y position with slight delay for realism
6. Smooth 60 FPS gameplay using requestAnimationFrame
7. Simple, clean visual design with a black background

Make the code well-commented and easy to understand for a beginner.
```

**What to Expect**:

Claude Code will generate a complete HTML file with:
- HTML structure with a canvas element
- CSS for styling
- JavaScript with game loop, physics, AI, and rendering

**Copy the Generated Code**:

1. Claude Code will respond with complete code
2. Copy the entire code block
3. Paste it into your `pong.html` file
4. Save the file

**Expected Code Structure** (Claude Code will generate something similar):

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Pong Game</title>
  <style>
    body {
      margin: 0;
      padding: 0;
      display: flex;
      justify-content: center;
      align-items: center;
      min-height: 100vh;
      background: #000;
      font-family: 'Courier New', monospace;
    }
    canvas {
      border: 2px solid #fff;
    }
  </style>
</head>
<body>
  <canvas id="pongCanvas" width="800" height="600"></canvas>

  <script>
    // Get canvas and context
    const canvas = document.getElementById('pongCanvas');
    const ctx = canvas.getContext('2d');

    // Game objects
    const paddleWidth = 10;
    const paddleHeight = 100;
    const ballSize = 10;

    // Player paddle (left)
    const player = {
      x: 20,
      y: canvas.height / 2 - paddleHeight / 2,
      width: paddleWidth,
      height: paddleHeight,
      dy: 0,
      speed: 6,
      score: 0
    };

    // AI paddle (right)
    const ai = {
      x: canvas.width - 30,
      y: canvas.height / 2 - paddleHeight / 2,
      width: paddleWidth,
      height: paddleHeight,
      speed: 4,
      score: 0
    };

    // Ball
    const ball = {
      x: canvas.width / 2,
      y: canvas.height / 2,
      dx: 5,
      dy: 3,
      size: ballSize,
      speed: 5
    };

    // Keyboard state
    const keys = {};

    // Event listeners for keyboard input
    window.addEventListener('keydown', (e) => {
      keys[e.key] = true;
    });

    window.addEventListener('keyup', (e) => {
      keys[e.key] = false;
    });

    // Draw rectangle (paddles)
    function drawRect(x, y, width, height, color = '#fff') {
      ctx.fillStyle = color;
      ctx.fillRect(x, y, width, height);
    }

    // Draw circle (ball)
    function drawCircle(x, y, radius, color = '#fff') {
      ctx.fillStyle = color;
      ctx.beginPath();
      ctx.arc(x, y, radius, 0, Math.PI * 2);
      ctx.fill();
      ctx.closePath();
    }

    // Draw text (scores)
    function drawText(text, x, y, color = '#fff') {
      ctx.fillStyle = color;
      ctx.font = '32px "Courier New"';
      ctx.textAlign = 'center';
      ctx.fillText(text, x, y);
    }

    // Draw center line
    function drawCenterLine() {
      ctx.strokeStyle = '#fff';
      ctx.setLineDash([10, 10]);
      ctx.beginPath();
      ctx.moveTo(canvas.width / 2, 0);
      ctx.lineTo(canvas.width / 2, canvas.height);
      ctx.stroke();
      ctx.setLineDash([]);
    }

    // Update player paddle
    function updatePlayer() {
      // W key or Arrow Up - move up
      if (keys['w'] || keys['W']) {
        player.y -= player.speed;
      }
      // S key or Arrow Down - move down
      if (keys['s'] || keys['S']) {
        player.y += player.speed;
      }

      // Keep paddle within canvas bounds
      if (player.y < 0) player.y = 0;
      if (player.y + player.height > canvas.height) {
        player.y = canvas.height - player.height;
      }
    }

    // Update AI paddle
    function updateAI() {
      // AI follows ball with slight delay for realism
      const aiCenter = ai.y + ai.height / 2;
      const ballCenter = ball.y;

      if (aiCenter < ballCenter - 20) {
        ai.y += ai.speed;
      } else if (aiCenter > ballCenter + 20) {
        ai.y -= ai.speed;
      }

      // Keep AI paddle within bounds
      if (ai.y < 0) ai.y = 0;
      if (ai.y + ai.height > canvas.height) {
        ai.y = canvas.height - ai.height;
      }
    }

    // Update ball
    function updateBall() {
      ball.x += ball.dx;
      ball.y += ball.dy;

      // Top and bottom wall collision
      if (ball.y - ball.size < 0 || ball.y + ball.size > canvas.height) {
        ball.dy = -ball.dy;
      }

      // Player paddle collision
      if (ball.x - ball.size < player.x + player.width &&
          ball.y > player.y &&
          ball.y < player.y + player.height) {
        ball.dx = Math.abs(ball.dx); // Ensure ball moves right

        // Add angle based on where ball hits paddle
        const hitPosition = (ball.y - (player.y + player.height / 2)) / (player.height / 2);
        ball.dy = hitPosition * 5;
      }

      // AI paddle collision
      if (ball.x + ball.size > ai.x &&
          ball.y > ai.y &&
          ball.y < ai.y + ai.height) {
        ball.dx = -Math.abs(ball.dx); // Ensure ball moves left

        // Add angle based on where ball hits paddle
        const hitPosition = (ball.y - (ai.y + ai.height / 2)) / (ai.height / 2);
        ball.dy = hitPosition * 5;
      }

      // Ball goes off left edge (AI scores)
      if (ball.x - ball.size < 0) {
        ai.score++;
        resetBall();
      }

      // Ball goes off right edge (Player scores)
      if (ball.x + ball.size > canvas.width) {
        player.score++;
        resetBall();
      }
    }

    // Reset ball to center
    function resetBall() {
      ball.x = canvas.width / 2;
      ball.y = canvas.height / 2;
      ball.dx = -ball.dx; // Reverse direction
      ball.dy = (Math.random() - 0.5) * 6; // Random Y velocity
    }

    // Render everything
    function render() {
      // Clear canvas
      drawRect(0, 0, canvas.width, canvas.height, '#000');

      // Draw center line
      drawCenterLine();

      // Draw paddles
      drawRect(player.x, player.y, player.width, player.height);
      drawRect(ai.x, ai.y, ai.width, ai.height);

      // Draw ball
      drawCircle(ball.x, ball.y, ball.size);

      // Draw scores
      drawText(player.score, canvas.width / 4, 50);
      drawText(ai.score, canvas.width * 3 / 4, 50);
    }

    // Game loop
    function gameLoop() {
      updatePlayer();
      updateAI();
      updateBall();
      render();
      requestAnimationFrame(gameLoop);
    }

    // Start game
    gameLoop();
  </script>
</body>
</html>
```

## Understanding the Code

Let's break down the key components of your game.

### Game Loop Architecture

```javascript
function gameLoop() {
  updatePlayer();   // Handle player input
  updateAI();       // Update AI behavior
  updateBall();     // Update physics
  render();         // Draw everything
  requestAnimationFrame(gameLoop); // Continue loop
}
```

**Why This Works**:
- `requestAnimationFrame` syncs with browser refresh (typically 60 FPS)
- Update logic before rendering ensures consistency
- Separation of concerns makes code maintainable

### Collision Detection

```javascript
// Paddle collision example
if (ball.x - ball.size < player.x + player.width &&
    ball.y > player.y &&
    ball.y < player.y + player.height) {
  // Collision detected!
}
```

**How It Works**:
- Checks if ball's left edge overlaps paddle's right edge (X-axis)
- Checks if ball's Y position is within paddle's height range
- Both conditions must be true for collision

### AI Behavior

```javascript
const aiCenter = ai.y + ai.height / 2;
if (aiCenter < ballCenter - 20) {
  ai.y += ai.speed;
} else if (aiCenter > ballCenter + 20) {
  ai.y -= ai.speed;
}
```

**How It Works**:
- Calculates center of AI paddle
- Compares to ball's Y position
- Moves toward ball with 20px "dead zone" for imperfect tracking
- Creates realistic, beatable AI

### Physics and Ball Movement

```javascript
ball.x += ball.dx; // Update X position
ball.y += ball.dy; // Update Y position

// Angle variation on paddle hit
const hitPosition = (ball.y - (player.y + player.height / 2)) / (player.height / 2);
ball.dy = hitPosition * 5;
```

**How It Works**:
- Simple velocity-based movement
- Hit position ranges from -1 (top of paddle) to +1 (bottom)
- Multiplying by 5 creates angled bounces
- Adds strategy - aim for paddle edges for sharp angles

## Playing Your Game

**Controls**:
- **W**: Move player paddle up
- **S**: Move player paddle down
- **SPACE**: Start game (on start screen)
- **P**: Pause/unpause
- **M**: Mute/unmute sounds
- **R**: Restart (on game over screen)

**Objective**:
- Hit the ball past AI paddle to score
- First to 5 points wins
- Use paddle edges to create sharp angles and trick AI

**Tips for Winning**:
- Position your paddle to hit ball at edges for sharp angles
- The AI has a small delay - use fast angles to your advantage
- Ball speeds up during rallies - prepare for faster gameplay

## Common Issues and Solutions

### Ball Moves Too Fast or Too Slow

**Solution**: Adjust ball speed in initial properties:

```javascript
const ball = {
  // ... other properties
  dx: 5,  // Lower = slower horizontal, higher = faster
  dy: 3   // Lower = slower vertical, higher = faster
};
```

### AI Too Hard/Easy

**Solution**: Adjust AI speed and dead zone:

```javascript
const ai = {
  // ... other properties
  speed: 4  // Lower = easier, higher = harder
};

// In updateAI function
if (aiCenter < ballCenter - 20) {  // Increase 20 = easier, decrease = harder
  ai.y += ai.speed;
}
```

### Ball Goes Through Paddle

**Problem**: Collision detection missed due to high speed
**Solution**: Lower ball speed or improve collision detection:

```javascript
// Ask Claude Code:
"The ball sometimes passes through paddles at high speeds. Improve
collision detection to handle high-velocity ball movement."
```

### Game Lags or Stutters

**Causes**:
1. Too many particles
2. Browser window not focused
3. Other applications consuming resources

**Solutions**:
- Reduce particle count
- Close other browser tabs
- Disable browser extensions while testing

### No Sound

**Check**:
1. Volume not muted (check M key toggle)
2. Browser sound enabled
3. Check browser console for Web Audio API errors
