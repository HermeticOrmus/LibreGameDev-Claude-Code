## Getting Help

### When You're Stuck

1. **Check the troubleshooting guide** - Most common issues are documented
2. **Search GitHub Discussions** - Someone may have had the same problem
3. **Ask Claude Code** - Use prompts like "Why isn't this collision detection working?"
4. **Ask the community** - Post in GitHub Discussions with your code and issue
5. **Review example code** - Compare your code to working examples

### Best Practices for Getting Help

When asking for help, provide:
- What you're trying to accomplish
- The prompt you used (if relevant)
- The code that isn't working
- What you expected vs. what happened
- Error messages (full text)
- What you've already tried

**Good Question Example**:
```
I'm trying to add collision detection to my Pong game following the
first-game-in-10-minutes.md tutorial. The ball passes through paddles
instead of bouncing. I used this prompt: [prompt]. Here's the collision
code: [code]. I expected the ball to bounce but it passes through.
I've tried adjusting the collision bounds but no change.
```

**Poor Question Example**:
```
My game doesn't work, help!
```
